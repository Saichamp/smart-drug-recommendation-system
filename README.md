# Smart Drug Recommendation System

An AI-powered clinical decision support tool that helps doctors, interns, and medical students recommend safe and appropriate medications based on patient symptoms, medical history, and lab reports.

## Project Overview

This system reduces prescription errors by:
- Analyzing patient data using machine learning
- Checking drug interactions and allergies
- Providing safety warnings for high-risk medications
- Supporting decision-making in small clinics and teaching hospitals

## Tech Stack

**Backend**: Python, Flask  
**Frontend**: HTML, CSS, JavaScript, Bootstrap  
**Machine Learning**: scikit-learn  
**Database**: SQLite (development), MySQL (production)

## Project Structure

backend/ - Flask API and ML model integration
frontend/ - User interface (HTML/CSS/JS)
data/ - Training datasets
models/ - Trained ML models
docs/ - Documentation and diagrams
